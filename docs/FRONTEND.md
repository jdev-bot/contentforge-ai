# Frontend Documentation

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Language:** TypeScript

## Project Structure

### Components

All reusable UI components are located in:

```
src/components/
```

### Pages

All Next.js pages and routes are located in:

```
src/app/
```

This project uses the Next.js 14 App Router pattern for routing and page organization.
