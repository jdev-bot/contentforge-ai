## What is the Smart Editor?

The Smart Editor is an AI-powered content transformation tool that uses advanced language models (Llama 3.3 70B via Groq) to:

- **Rewrite** - Change tone and style while keeping the message
- **Expand** - Add depth, examples, and detail
- **Condense** - Create shorter versions preserving key points
- **Optimize** - Tailor content for specific platforms

## Step 1: Access the Smart Editor

### Navigation Options

**Option 1: From Content**
1. Go to **Content**
2. Select a content item
3. Click **"Open in Editor"**

**Option 2: Direct Access**
1. Click **"AI Editor"** in navigation
2. Paste or type your content

**Option 3: From Assets**
1. View a generated asset
2. Click **"Edit with AI"**

## Step 2: Rewrite Content

### When to Use Rewrite

- Change tone for different audiences
- Adapt style for different platforms
- Refresh old content
- Create variations for A/B testing

### Using Rewrite

1. Enter or paste your content:
   ```
   Our company provides excellent software 
   development services. We help businesses 
   build scalable applications.
   ```

2. Select **"Rewrite"** tab

3. Choose **Tone**:
   ```
   [ ] Casual        [ ] Professional
   [x] Witty        [ ] Formal
   [ ] Friendly     [ ] Authoritative
   [ ] Enthusiastic [ ] Empathetic
   ```

4. Choose **Style**:
   ```
   [ ] Neutral      [x] Persuasive
   [ ] Informative  [ ] Storytelling
   [ ] Concise      [ ] Descriptive
   ```

5. Click **"Generate Rewrite"**

### Example Output

**Input (Professional + Neutral):**
```
Our company provides excellent software 
development services. We help businesses 
build scalable applications.
```

**Output (Witty + Persuasive):**
```
We're the code wizards your business has been 
waiting for. While others promise the moon, 
we actually deliver scalable applications 
that won't crash when your traffic explodes.
```

### Tone + Style Combinations

| Tone | Style | Best For |
|------|-------|----------|
| Professional | Informative | B2B content, reports |
| Casual | Conversational | Social media, blogs |
| Enthusiastic | Persuasive | Product launches, sales |
| Authoritative | Concise | Thought leadership |
| Friendly | Storytelling | Newsletters, community |

## Step 3: Expand Content

### When to Use Expand

- Turn ideas into full articles
- Add examples and case studies
- Create comprehensive guides
- Elaborate on key points

### Using Expand

1. Enter brief content:
   ```
   We help businesses grow through digital marketing.
   ```

2. Select **"Expand"** tab

3. Set **Expansion Target**:
   ```
   Target: 3x length
   ```

4. Add **Focus Areas** (optional):
   ```
   Focus: Strategies, Case studies, ROI examples
   ```

5. Click **"Generate Expansion"**

### Example Output

**Input:**
```
We help businesses grow through digital marketing.
```

**Output (3x expansion):**
```
At our core, we specialize in helping businesses 
achieve remarkable growth through comprehensive 
digital marketing strategies. Our approach combines 
data-driven insights with creative execution to 
deliver measurable results.

For example, one of our clients, a mid-sized 
e-commerce company, saw a 340% increase in organic 
traffic within six months of implementing our 
SEO and content marketing strategy. By targeting 
high-intent keywords and creating valuable content,
they went from 5,000 monthly visitors to over 22,000.

Our services include:
• Search engine optimization (SEO)
• Pay-per-click advertising (PPC)
• Social media marketing
• Content creation and strategy
• Email marketing campaigns

The ROI of digital marketing speaks for itself. 
On average, our clients see a 4:1 return on their 
marketing investment within the first year.
```

### Expansion Tips

- Start with clear main points
- Use focus areas to guide AI
- Review for accuracy of examples
- Edit to match your experience

## Step 4: Condense Content

### When to Use Condense

- Create executive summaries
- Make TL;DR versions
- Fit content into character limits
- Extract key takeaways

### Using Condense

1. Enter long content:
   ```
   Our comprehensive guide to digital marketing 
   covers everything from SEO to social media 
   advertising. We dive deep into keyword research, 
   content strategy, paid campaigns, and analytics. 
   Learn how to optimize your website for search 
   engines, create engaging social media content, 
   run effective ad campaigns, and measure your 
   results using advanced analytics tools...
   ```

2. Select **"Condense"** tab

3. Set **Target Percentage**:
   ```
   Target: 40% of original
   ```

4. Check **Preserve Key Points**

5. Click **"Generate Summary"**

### Example Output

**Input (100 words)**

**Output (45 words - 45% reduction):**
```
Master digital marketing with our guide covering 
SEO, social media, paid campaigns, and analytics. 
Learn optimization, content creation, ad management, 
and results measurement.
```

### Condense Options

| Target | Use Case |
|--------|----------|
| 80% | Minor tightening |
| 60% | Executive summary |
| 40% | TL;DR version |
| 20% | Tweet/post length |

## Tips for Best Results

### Input Quality

✅ **Good Input:**
- Clear, complete sentences
- Proper grammar and spelling
- Single topic focus
- Adequate length (100+ words for best results)

❌ **Avoid:**
- Bullet lists without context
- Single sentences
- Typos and errors
- Mixed topics

### Editing Workflow

**Recommended Order:**
1. Start with your best draft
2. Expand if needed (add detail)
3. Rewrite for tone/style
4. Optimize for platform
5. Condense if too long

### Token Usage

| Operation | Typical Tokens |
|-----------|----------------|
| Rewrite | 200-500 |
| Expand | 500-2000 |
| Condense | 200-400 |
| Optimize | 300-600 |

Monitor your monthly usage in the dashboard.

## Troubleshooting

### "Content too short"

**Minimum recommended:** 50 characters

**Fix:**
- Add more context
- Combine multiple points
- Use Expand first

### "Output doesn't match my brand"

**Fix:**
- Try different tone combinations
- Manually edit AI output
- Set brand voice in project settings
- Save preferred settings as templates

### "Generated content has errors"

**Important:** AI can make mistakes.

**Always:**
- Fact-check claims
- Verify statistics
- Review for accuracy
- Edit for your voice
